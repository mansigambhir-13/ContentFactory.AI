## ?? Quick Start

1. **Install dependencies:**
   `ash
   pip install -r requirements.txt
   `

2. **Set up environment:**
   - Copy .env template and add your OpenAI API key
   - Optionally add webhook URLs for automation

3. **Run the engine:**
   `ash
   python free_posting_main.py
   `
